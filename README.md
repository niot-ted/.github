# NIoT TED Proof of Concept OpenSAFELY

This organistion is part of the proof of concept implementation of the OpenSAFELY model in the NIoT's Teach Education Dataset (TED) backend.

It contains
 - repos that hold the research code that will be run on the senstive TED data
 - the users who are allowed to run the above code are members of this organisation.

For more background see: https://schools.opensafely.org/
